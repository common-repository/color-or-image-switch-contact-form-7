== Description ==
**Color or Image Switch Contact Form 7** is a clean, simple to use elegant option with Switch to setup in contact form 7 for your [WordPress](https://en.wikipedia.org/wiki/WordPress) site.

&#128312; **[Demos](http://oceanwebguru.com/switch-demo/)**

**Contact form 7 image option** allow to setup image as option and setup value as form.

Color Radio button can be setup with show good idea for customer who want to do enquiry for option.
